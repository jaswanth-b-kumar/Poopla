# Mobile App

React Native (Expo) client for Poopla.

## Prerequisites
- Node.js 18+ (or 20+)
- npm
- Expo CLI

## Setup
```bash
cd mobile
npm install
```

## Run (dev)
```bash
npm run start
```

## Notes
This folder is a scaffold. Run `npx create-expo-app .` to initialize the Expo app
or tell Codex to generate a minimal Expo project here.
